# Генератор парольей

Генератор парольей создан с HTML, CSS и Javascript

## Как он использовать

Все вы должны делать загрузить папка и вы готовы. Откройте index.html, выбирайте параметры как вы хотите а потом просто нажмите на кнопке "Generate !".
